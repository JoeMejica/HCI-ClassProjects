Part 2A

Project 2 will be designed to support the Atlanta Filipino Student Association’s annual performance showcase “Culture Shock” The Atlanta Filipino Student Association (ATL FSA) is a student organization comprised of students from Georgia State, Georgia Tech, and other surrounding universities who want to gain a better understanding and appreciation of the Filipino culture.  ATL FSA is involved in many events such as general body meetings, community service, Def Talent Jam dance competition, and Culture Shock. Through these events, ATL FSA strives to represent the organization as a strong, talented, and diverse organization. Culture Shock, in which the website will be centered around, is ATL FSA’s premier event held each Spring semester at Georgia State University. The event is primarily a talent showcase, exhibiting performances from organization members, surrounding performing groups, and other collaborative student organizations.

The projected user community for the website is directed towards college students in the surrounding Atlanta area universities such as Georgia State, Georgia Tech, SCAD, and AUC. Because the projected user community is centered around the college demographic, the age range in the user community is projected to be 18-25 years of age for both males and females. In addition to the basic demographical information, the following interest groups are projected to play factor in the user community for the website: Filipino, Asian-American culture, Performing Arts, Food, Music, Dance, and Georgia State University.

The website will be designed for users to learn more about the Culture Shock event and how the event is a grand aspect of ATL FSA. Prior to the date of the event, the website will include general event info such as the date, time, location of Culture Shock, history of Culture Shock, a social media media feed via twitter for users to keep up-to-date about the event, and links on how users can either perform for the event and/or financially support Culture Shock as a sponsor. Goals and objectives that the website has for the user include having a better understanding of the event and how it relates to ATL FSA, obtain interest in following ATL FSA via social media to learn more, and obtain interest in attending the event.

As mobile computing continues to grow exponentially, the fundamental focus in regards to designing the website is “mobile-first”. This can be achieved by creating a fully functioning website that is not only reliable but highly user friendly. Aspects such as easy to read content, interactive web elements, and quick responsiveness will assist in helping users achieve the goals and objectives mentioned above. When the user accesses the simple URL, they will be treated to an appealing website where they will not only be able to access and learn all the information that desired and more, but will be more inclined to share the site with others. Thus will this in mind, Project 2 will not only be aimed at supporting ATL FSA and Culture Shock, but to support Human Computer Interaction through delivering content and information in an aesthetically and technically leveled method.
